Data

The concatenated dataset is a large version, which combines the datasets of all 28 EU member state countries. The data is collected from national and regional portals of the member states where the beneficiary lists are published. We have collected 115 datasets, 21 datasets are missing. This means that for most countries, we have identified all data for all Structural Funds. However for Ireland, UK, Spain, and Austria we still have major gaps.

The original files are described here For creating the joined datasets out of these files, we mapping every single datasets to common denominator variables that can be compared between the countries. You can find a list of all variables used below. More information on the methods can be found in the Methodology section.

Concatenated Dataset

List of Variables:

  • beneficiary_name is name of the beneficiary (person, company, organisation)
  • project_name is name of project
  • project_description is description of the project
  • project_id is unique code of the project (generated by authority itself)
  • beneficiary_person is name of person responsible
  • project_status is status of the project
  • starting_date is starting date of the project
  • completion_date is completion date of the project
  • approval_date is approval date of the project
  • final_payment_date is date on which the final payment was made
  • theme_name is name of the thematic objective
  • theme_code is code of the thematic objective
  • cci_program_code is CCI codes identifying operational programs
  • priority_label is description of the priority number of the grant agreement
  • priority_number is priority number of the grant agreement
  • management_authority is management authority
  • operational_programme info on which programme is governing the project
  • total_amount is total cost of project
  • total_amount_eligible is total eligible expenditure
  • member_state_amount is amount that is awarded from national funds
  • eu_cofinancing_amount is amount of co-financing from the EU
  • eu_cofinancing_amount_eligible is amount of co-financing a project is eligible for
  • eu_cofinancing_rate is rate (percent) of co-financing from the EU
  • third_party_amount is total amount additional to the action over third party funding
  • fund_acronym is acronym of the fund (ERDF, ESF, CF)
  • beneficiary_address is full address of the beneficiary
  • beneficiary_city is city of beneficiary
  • beneficiary_postal_code is postal code of beneficiary
  • beneficiary_nuts_region is region matching the NUTS code
  • beneficiary_nuts_code is NUTS code of beneficiary region
  • beneficiary_county is county of beneficiary
  • beneficiary_country is country of beneficiary
  • beneficiary_country_code is two digit NUTS country code of beneficiary (see github for the list)
  • beneficiary_url is URL of the project
  • source is a source url of the original data
